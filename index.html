Veelgestelde vragen

Is dit een serieuze website?

Nee. Dit is een parodie op de Belgische beslissing om kerncentrales op te kopen, geïnspireerd door wijkopenautos.be. De kostprijzen zijn gebaseerd op publiek beschikbare schattingen, maar de website is satirisch van aard.

Zijn de bedragen realistisch?

De totaalbedragen zijn gebaseerd op publieke schattingen van de deal met Engie (ca. €15 miljard voor Doel 4 en Tihange 3). De persoonlijke aandelen worden berekend op basis van 5,5 miljoen Belgische belastingbetalers. Dit zijn ruwe benaderingen.

Kan ik echt een kerncentrale kopen?

Nee. Kerncentrales zijn niet te koop voor particulieren. De Belgische staat heeft dit al voor u gedaan. U betaalt enkel de rekening.

Wat is open source aan deze website?

De volledige broncode is beschikbaar op GitHub onder een MIT-licentie. Iedereen mag de code hergebruiken, aanpassen of verbeteren. In tegenstelling tot de deal met Engie is hier volledige transparantie.

Wie heeft dit gemaakt?

Dit is een burgerproject, gemaakt door iemand die zijn belastingbrief goed heeft gelezen. Geen politieke partij of organisatie zit hierachter.
